# firstsit-tools

Free FINRA/NASAA exam-prep tools: pass-mark calculators, glossaries, and free practice quizzes
for SIE, Series 6, 7, 63, 65, 66. Built and deployed from the main `firstsit` repo's
`06_SITE/build_site.py` — this repo just hosts the static output via GitHub Pages.

Live at https://jayblip.github.io/firstsit-tools/
